HealthComponent

case class HealthComponent(name: String, state: ComponentState = ComponentState.NORMAL, details: String, extra: Option[AnyRef] = None, components: List[HealthComponent] = List.empty) extends JsonSerializable with Product with Serializable

The health component is used to define a subset of functionality that can be asked of it's state. It can contain optional child components.
