SimpleHttpServer

class SimpleHttpServer extends HActor with ComponentHelper

This is a simple http server, which is primarily started for HealthChecks And will have basic CIDR rules applied to it by default, but will be configurable. This basic Http handler should not be extended to any functionality outside of base functionality for the core server. For that use one of the Http Components like Spray, Socko or Netty.
